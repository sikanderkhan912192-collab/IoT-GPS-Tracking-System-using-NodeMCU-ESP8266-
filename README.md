# 📍 IoT GPS Tracking System

## 🚀 Overview

This project is a real-time GPS tracking system built using NodeMCU (ESP8266) and Neo-6M GPS module. It provides live location tracking using Google Maps.

## 🛠️ Features

* Real-time GPS location tracking
* Google Maps integration
* IoT-based system design
* Expandable to web/mobile dashboard

## 📦 Components

* NodeMCU (ESP8266)
* Neo-6M GPS module
* Jumper wires

## ⚙️ Working

The GPS module collects location data (latitude & longitude), which is processed by NodeMCU and converted into a Google Maps link.
